Our Objective

Creating Visual representation has great potential to enhance learning and teaching throughout the many stages involved from researching pedagogical practice, scholarship, linking research and teaching, planning and curriculum development through to presentation and evaluation amongst many others.

Whilst images can arguably provide a visual enhancement to text heavy forms of communication, they can also convey meanings and impact emotions. In addition, images can enhance the learning experience and provide for a more inclusive approach to learning and teaching.

The creation of suitable visual materials helps the author to structure and make sense of their own thinking. The process helps “embed representations, using graphical and textual semiotic conventions of their creator’s understanding of a given issue”.

In this project we have tried to make those visual representations even easier to create for your various uses. With the help of math, logic and CSS and JavaScript we have managed to create a website which makes charts which in our opinion is the most appealing visual representation.

Our Methodology

TO CREATE GRAPH:
  • Take input of data from user in form of numbers or percentages.
  • Calculate area of chart respective to each data member.
  • Display chart with calculated values.
TO ACCEPT DESCRIPTIONS:
  • Let user select required portion of graph.
  • Show a text box to write description.
  • Show different font and other text formatting options.
  • Save description after user clicks button.
TO SHOW DESCRIPTIONS:
  • Let user select required portion of graph.
  • Show text box with description present.
  • Allow user to edit description if desired.
TO EDIT DATA:
  • Let user select required portion of graph.
  • Show pre-existing value of data member.
  • Accept new value of member.
  • Recalculate graph area and display new graph.
